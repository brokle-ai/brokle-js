# Brokle Platform JavaScript SDK

A comprehensive JavaScript SDK for the Brokle Platform that provides intelligent routing, cost optimization, semantic caching, and comprehensive observability for AI applications.

## Coming Soon

The JavaScript SDK is currently in development. Stay tuned for:

- Intelligent routing and semantic caching
- Real-time analytics and evaluation

## Installation

```bash
npm install @brokle/sdk
```
